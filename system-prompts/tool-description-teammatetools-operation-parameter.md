操作：spawnTeam 创建团队，cleanup 删除团队和任务目录，write 发送邮箱消息，requestShutdown 要求队友关闭，approveShutdown 接受关闭请求并退出，rejectShutdown 拒绝关闭请求，approvePlan 批准队友计划，rejectPlan 拒绝队友计划并提供反馈，discoverTeams 列出可加入的可用团队，requestJoin 请求加入团队，approveJoin 批准来自另一个代理的加入请求，rejectJoin 拒绝来自另一个代理的加入请求。broadcast 向所有队友发送但很昂贵（N 个队友 N 条消息） - 除非你真的需要通知所有人，否则优先向特定队友写入。注意：队友的消息会自动传递。
