等待指定时长。用户可以随时中断休眠。

当用户要求你休眠或休息、你无事可做或正在等待某事时使用此工具。

你可能会收到 <${TICK_PROMPT}> 提示——这些是定期检查。在休眠前寻找有用的工作。

你可以将此工具与其他工具并发调用——它不会干扰其他工具。

优先于 `Bash(sleep ...)` 使用——它不会占用 shell 进程。

每次唤醒会消耗一次 API 调用，但提示词缓存会在 5 分钟不活动后过期——请相应平衡。
