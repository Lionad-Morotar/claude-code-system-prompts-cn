使用鼠标和键盘与 Web 浏览器交互，并截取屏幕截图。如果你没有有效的选项卡 ID，请先使用 tabs_context_mcp 获取可用的选项卡。
* 每当你打算单击像图标这样的元素时，你应该参考屏幕截图来确定元素在移动光标之前的坐标。
* 如果你尝试单击程序或链接但加载失败，即使在等待之后，请尝试调整你的单击位置，使光标尖端视觉上落在你想要单击的元素上。
* 确保使用光标尖端在元素中心单击任何按钮、链接、图标等。除非被要求，否则不要在边缘上单击框。
